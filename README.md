# SnowRunner Speedometer/Distance Traveled HUD + Time Control Mod

Combined mod that provides speed/distance tracking and time control functionality in a single .asi file.

## Features

### Speed & Distance HUD
- Real-time speed display in km/h
- Distance tracking in meters
- Customizable HUD position and scale
- Toggle speed/distance display independently
- Reset distance counter

### Time Control
- **Freeze Time** - Stop the in-game day/night cycle
- **Timelapse Mode** - Rapidly cycle through time for screenshots/videos
- Current time display when frozen

## Installation

1. Download and place Ultimate ASI Loader (dinput8.dll) in your SnowRunner game directory "Snowrunner\Sources\Bin"
2. Place `dinput8.dll` in your SnowRunner game directory "Snowrunner\Sources\Bin"
3. Place the `speedhud.asi` file in your SnowRunner game directory "Snowrunner\Sources\Bin"
4. Run the game

## Controls

### HUD Controls (Menu must be open - Press HOME to toggle menu)
- **Arrow Keys**: Move HUD position
- **Ctrl + +/-**: Scale HUD size
- **Ctrl + 0**: Toggle speed display
- **Ctrl + 9**: Toggle distance display
- **Ctrl + 8**: Reset distance counter
- **Ctrl + S**: Save HUD settings

### Time Controls (Work anytime)
- **F1**: Toggle freeze time on/off
- **F2**: Toggle timelapse mode on/off

### General
- **HOME**: Toggle settings menu
